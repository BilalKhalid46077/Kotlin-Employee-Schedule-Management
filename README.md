# Installation Guideline

#Install latest version of Android Studio

#Install emulator on Android Studio or connect your android device with datacable.

#Clone Repo SSH "git@github.com:BilalKhalid46077/Kotlin-Employee-Schedule-Management.git"

#Open Android Studio go to File->Open->Locate repo directory -> open folder Kotlin-Employee-Schedule-Management and click open

#Android Studio will load project

#Click on green "Run app" button

#App will run on emulator/android device
